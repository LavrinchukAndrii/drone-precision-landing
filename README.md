# Система точної посадки дрона (Precision Landing)

Цей проєкт реалізує політ дрона в симуляторі ArduPilot за допомогою Python та бібліотеки DroneKit.

### Основні фішки:
* **ПІ-регулятор швидкості**: Дрон активно бореться з вітром 5 м/с.
* **Фаза Precision Descent**: При підльоті до цілі (4.5м) вмикається плавне зниження.
* **Точність**: Досягнуто стабільного приземлення з похибкою ~1.1 метра при спуску з 300м.

### Як запустити:
1. Запустити SITL симулятор через Mission Planner.
2. Встановити залежності: `pip install dronekit`.
3. Запустити скрипт: `python fly.py`.
